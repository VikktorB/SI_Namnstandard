# In/Utgångar

## Namnstandard

PREFIX_SYSTEM_NAMN

Exempel på namn till en varmvattenkrets:

AI_VV1_GT_TILL_1
AI_VV1_GT_RETUR_1
AO_VV1_SV_1
DO_VV1_P1
DI_VV1_P1

## PREFIX_

| Prefix | Beskrivning |
| --- | --- |
| AI_ | Analog ingång |
| DI_ | Digital ingång |
| AO_ | Analog utgång |
| DO_ | Digital utgång |

## _SYSTEM (exempel)

| System | Beskrivning |
| --- | --- |
| VSx | Värmesystem |
| VVx | Varmvatten |
| LBx | Luftbehandling |

## _NAMN (exempel)

| Namn | Beskrivning |
| --- | --- |
| _TILL | Tillopp/Tilluft |
| _RETUR | Retur |
